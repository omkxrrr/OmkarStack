# OmkarStack

OmkarStack is a personal portfolio website for Omkar Mede, showcasing skills, projects, experience, social profiles, and contact details in a single-page interactive design.

## Highlights

- Single-page portfolio layout
- Smooth navigation between sections
- Animated project cards
- Featured projects: TaskFlow and AageKyaKaru
- Responsive design for desktop and mobile

## Run Locally

Open `index.html` directly in a browser.
